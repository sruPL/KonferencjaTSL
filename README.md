# II Pomorskie Dni Logistyki - strona

Statyczna strona konferencji publikowana przez GitHub Pages.

## Najważniejsze pliki

- `index.html` - strona główna.
- `kompasy/index.html` - podstrona `Słupski Kompas Logistyki`.
- `CNAME` - domena GitHub Pages, obecnie `www.konferencjatsl.pl`.

## Assets

- `assets/css/index.css` - style strony głównej.
- `assets/css/kompasy.css` - style podstrony Kompasy.
- `assets/js/index.js` - nawigacja i przewijanie na stronie głównej.
- `assets/js/kompasy.js` - przejścia z podstrony Kompasy do sekcji strony głównej.
- `assets/img/hero/` - obraz hero.
- `assets/img/logos/` - logotypy organizatorów, patronów i mediów.
- `assets/img/partners/` - logotypy partnerów.
- `assets/docs/kompasy/` - dokumenty konkursowe do pobrania.
- `assets/archive/` - stare pliki archiwalne/backup.

## Kompasy i dokumenty

Na podstronie `kompasy/index.html` linki do dokumentów konkursowych są obecnie zakomentowane. Po odkomentowaniu wskazują na:

- `assets/docs/kompasy/wniosek-zloty-kompas-biznesu.docx`
- `assets/docs/kompasy/wniosek-zloty-kompas-edukacji.docx`
- `assets/docs/kompasy/regulamin-zloty-kompas-biznesu-i-edukacji.docx`

## Uwaga przy zmianach

- Przy przenoszeniu plików zawsze popraw ścieżki w HTML/CSS.
- Linki zewnętrzne otwierane w nowej karcie powinny mieć `rel="noopener noreferrer"`.
